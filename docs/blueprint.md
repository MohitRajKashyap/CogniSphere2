# **App Name**: CogniSwap

## Core Features:

- User Registration: Allow users to register with their name, skills they can teach, and skills they want to learn.
- Skill Matching: Match users based on the overlap between skills they can teach and skills they want to learn using a basic matching algorithm.
- Credit System: Implement a credit system where users earn credits for teaching and spend credits for learning.
- AI Learning Module (Text): Generate beginner-friendly explanations and key concepts for a selected skill using Google Gemini API.
- AI Learning Module (Code): Provide real, runnable code examples with comments and mini-tasks for practical application, powered by Google Gemini API.
- AI Learning Module (Audio/Visual Simulation): Generate an 'Audio Narration Script' and 'Visual Explanation' text using Google Gemini API to simulate audio-visual learning content.
- Session Evaluation: Evaluate user understanding based on a session summary, providing a score, feedback, and improvement tips using Google Gemini API as a tool.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a calm, intellectual feel.
- Background color: Very light grayish-blue (#E8EAF6).
- Accent color: Vivid violet (#7C4DFF) to highlight key UI elements.
- Body and headline font: 'Inter', a sans-serif, for a modern and neutral look, appropriate for body and headlines
- Use flat, outlined icons to represent different skills.
- Dashboard-style layout with clear sections for skill selection, learning modules, and credit display.
- Subtle animations to provide feedback on interactions (e.g., button presses, loading AI content).
